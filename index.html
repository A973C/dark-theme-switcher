Introduction Last updated: 2020-08-07

Theme Inverter is a tiny Js Library which allows user to change the theme of website.

Theme Inverser Example:

You can use Theme Inverser by instantiating it 

window.ThemeSwitcher().Init();

Installation

Installing Theme Inverser is easier than using it

Copy and Paste the following code in your <head> tag in webpage 

<!-- development version, includes helpful console warnings --><
  <script src="https://binayak.codes/dark-theme-switcher/app.js"></script>
 <link rel="stylesheet" href="https://binayak.codes/dark-theme-switcher/styles/style.css">

And you are done !!!

Additional Features

The glowy effect in theme toggler is optional

To add this glowy effect. You have to provide config while instantiating it. 

window.ThemeSwitcher().Init({glow:true});

And you can get this awesome glowy animation!!!
